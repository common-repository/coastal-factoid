== Description ==

The Coastal Factoid plugin provides a Wordpress Widget that displays the daily (M-F) Coastal Factoid from the Surfrider Foundation's Beachapedia coastal knowledge encyclopedia. Coastal Factoids are also sent out via Twitter by the 'coastalfactoid' twitter account.

== Installation ==

1. Upload and activate the plugin (coastalfactoid.zip)
1. Drag the Coastal Factoid widget to the place in the theme that you would like the widget to appear

== Frequently Asked Questions ==

= Who curates the factoids? =

Factoids are curated by the environmental staff at the Surfrider Foundation

= What happens if the factoid is unavailable? =

The Widget uses AJAX to load the factoid asynchronously, so the rest of the page should experience no delay in rendering.
